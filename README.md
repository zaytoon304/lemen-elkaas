# 🏆 لمن الكأس؟

لعبة مسابقات تعليمية للفرق — مناسبة للفصول الدراسية والمنزل.

## الميزات
- **6 مواد**: رياضيات، عربي، إنجليزي، علوم، اجتماعيات، تربية إسلامية
- **6 صفوف دراسية** (الأول → السادس) × **3 مستويات** (سهل / متوسط / صعب)
- **1080 سؤال** متعدد الخيارات (10 سؤال لكل صف × مستوى × مادة)
- **مؤقت 30 ثانية** لكل سؤال
- **فرصة الاختطاف**: إذا أخطأ فريق أُعطي الآخر 15 ثانية للاختطاف
- تأثيرات بصرية وكونفيتي احتفالي عند الفوز

## طريقة اللعب
1. اكتب أسماء الفريقين
2. اختر المادة والصف والمستوى
3. ابدأ المسابقة — الفريقان يتناوبان على الإجابة
4. الفريق الفائز بأعلى نقاط يحصل على الكأس 🏆

